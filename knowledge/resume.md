# Resume Summary

## Current Role

Rudy Pena is currently a Staff Software Engineer, AI Platform at DTN. He has been at DTN since March 2022.

At DTN, Rudy designs secure, identity-aware AI platform infrastructure on AWS. His work enables teams to prototype and deploy agent-driven workflows with shared and custom MCP tooling.

His DTN work includes runtime and deployment patterns for AWS Bedrock AgentCore, Strands-based agents, and FastAPI MCP services using API Gateway, WAF, Lambda authorizers, ECR, S3, Aurora/Postgres, and OpenSearch.

Rudy created Python CLI and SDK tooling to bootstrap agent services, standardize configuration, and reduce time-to-prototype for customer-facing AI use cases.

He also delivered product and platform surfaces in Next.js for agent creation, configuration, and interaction across engineering and product teams.

He has delivered production AI features with AWS Bedrock Claude and Titan for summarization, analysis, and insight generation in DTN's Grain Intelligence platform.

Rudy worked as a Software Engineer at Bitfusion from September 2017 to August 2018. He built Go CLI tooling for distributed machine learning workloads with Cobra and Viper. He also developed full-stack features with Go, Node.js, and React for managing ML jobs and clusters.
