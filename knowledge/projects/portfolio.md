# Personal Portfolio

Rudy's personal portfolio is a Next.js site with themeable design tokens, a terminal-inspired interaction layer, and content shaped around applied AI, developer tooling, and product craft.

The terminal is intended to be a deliberate feature, not just decoration. It supports commands like `help`, `about`, `skills`, `stack`, `work`, `projects`, `current`, `contact`, `resume`, and navigation commands like `open projects`.

The Ask Rudy roadmap extends the terminal into a retrieval-backed AI interface. The terminal should remain the command surface, while cards render answer summaries, evidence, role-fit analysis, source chunks, and lab/debug details.

This project demonstrates Rudy's interest in tasteful interaction design, terminal UI, Next.js, TypeScript, responsive theming, and interfaces that make technical ideas approachable.
