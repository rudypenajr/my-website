# Time Crisis Wiki

Time Crisis Wiki is Rudy's retrieval app for Time Crisis episode data.

The project is backed by a Go scraper, MongoDB Atlas, OpenAI embeddings, vector search, and a React interface for episode search and AI-assisted exploration.

It is a practical retrieval application that connects data collection, storage, embeddings, vector search, and a user-facing search interface.

The project is useful evidence for Rudy's interest in RAG, data pipelines, vector search, embeddings, MongoDB Atlas, Go, and React.
