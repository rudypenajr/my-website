# Wavves - Music Lab

Wavves is Rudy's music AI lab for concert prep, smart playlists, and song discovery.

It combines Setlist.fm, Spotify, FastAPI inference, speech-to-text, query understanding, transformer classifier experiments, and evaluation dashboards.

The project is designed to make music intent observable and useful. It explores how product workflows, inference services, and evaluation loops can work together in a practical AI application.

Wavves includes work around song recognition, playlist generation, speech-to-text, query understanding, deterministic workflow orchestration, and eval baselines.

The project uses Next.js, FastAPI, STT, Deepgram, Spotify, Inngest, transformer classifier experiments, and ML inference patterns.
